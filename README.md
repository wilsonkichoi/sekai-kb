# LagunaBeach.md

A knowledge base for Laguna Beach, CA — plain Markdown content rendered as a fast,
AI-native static site. Built with [Astro](https://astro.build/); the knowledge base
lives as Markdown under `knowledge/` and is the single source of truth (`src/content/`
is derived and gitignored).

This is also **instance #1 of the `sekai-kb` framework** — a reusable template for
place knowledge bases that gets cut from this tree at Phase 5.

> **🚧 Work in progress.** This is an active rebuild. Structure, content, and tooling
> are still landing phase by phase; expect things to move.
